Research Semester 2018-2019

In this repository the files of my research about the esp32.
This research is made for my study Computer Engineering at the Hogeschool Utrecht.
In this research I've tried to make my own build system for this microcontroller using the libraries and tools from the Hogeschool Utrecht.


The repository consists of the following components:
 - Deliverables: The documents made and used for the research
 - ti-software: The modified components of libraries and tools form the Hogeschool Utrecht
 - Workspace: Testing area of the made software
